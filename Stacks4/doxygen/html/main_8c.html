Detailed Description

Reversing a linked list using stacks. LIFO. The time complexity of stack is Big O(1). The Linked list that inserts a list at the end of the list has a time complexity of Big O(n) This is because it has to iterate through the list until the end to add one. However, the linked list that inserts a list at the beginning of the list does not go through this iteration. Thus, the time complexity is Big O(1), same as the time complexity of stack.
