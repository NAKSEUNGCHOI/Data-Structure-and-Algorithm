Detailed Description

Parentheses balance checker using stacks. LIFO. Stack is a suitable data structure for parentheses balance checker. As an opening parentheses comes in, it pushes the parentheses into stack. In contrast, it pops the opening parentheses as the corresponding closing parentheses comes in.
